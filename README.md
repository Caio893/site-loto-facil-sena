# Sistema de Conferência de Resultados de Loteria

Aplicação web para conferência automática de resultados da Mega-Sena, Lotofácil e Quina.

## Funcionalidades
- Consulta de resultados
- Comparação automática de jogos
- Cálculo de acertos
- Interface web simples

## Tecnologias
- JavaScript
- HTML
- CSS
